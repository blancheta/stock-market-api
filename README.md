# stock-market-api

### Create virtual environment
```
python -m venv venv
```
### Activate virtual environment
```
source venv/bin/activate
```

### Install the dependencies
```
pip install -r requirements.txt
```

### Run the project
```
export FLASK_APP=app
export FLASK_ENV=development
flask run
```

### Run the tests

```
python -m pytest -vv
```
